Prompt para corrigir erro de url_for no ActiveAdmin:

1. Abra o arquivo onde ocorre o erro (ex: app/admin/categories.rb).
2. No topo do arquivo (antes do ActiveAdmin.register ...), adicione:
   include Rails.application.routes.url_helpers
3. Se o erro persistir, adicione também:
   include ActionView::Helpers::UrlHelper
   include ActionView::Helpers::AssetUrlHelper
4. Salve o arquivo e reinicie o servidor Rails.
5. Recarregue a página do ActiveAdmin.

Resumo:
